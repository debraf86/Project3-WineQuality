Predicting Wine Quality


The purpose of this project was to analyze the quality of red wine based on several attributes such as alcohol, density, sulfates, pH, etc. We will then use this data in machine learning to predict the quality of a wine. This dataset is from vineyards in Portugal and was retrieved from Kaggle.com
